## Overview

The **Eiger Career Page** is an end-to-end web application developed to streamline the company's hiring process. The platform serves two distinct primary users: prospective employees and internal HR administrators. 

For applicants, it provides a seamless _user-facing portal_ to explore career opportunities, read the company's privacy policies, and apply for open positions. For internal teams, it includes a powerful **Content Management System (CMS)** designed to oversee the entire recruitment pipeline, track applicants, manage submitted documents, and control administrative user access.

## Key Contributions & Technical Achievements

Throughout the development cycle of the Eiger Career Page, I functioned as a versatile developer, actively contributing to both the public portal and the internal CMS. My primary responsibilities included:

- **_Front-End_ Component Development**:
  Built the core _front-end_ components for the public career page, including the Homepage, FAQ section, and Privacy Policy. I utilized a modular and clean code approach to ensure the interface was highly responsive, maintainable, and aligned with the brand's identity.
- **_CMS_ Back-Office Features**:
  Designed and developed critical CMS features for the HR team. This included building comprehensive lists and detailed view pages for applicant document submissions.
- **API Integration & Data Flow**:
  Handled the complex API integrations bridging the _front-end_ UI with the _backend_ services. Ensuring smooth, secure, and rapid data flow to deliver a responsive user experience.
