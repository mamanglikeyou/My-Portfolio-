## 📖 Overview

---

This project is an implementation of an automatic Indonesian paraphrasing system using the IndoT5 (Text-to-Text Transfer Transformer) model.
The system is designed to generate new sentences with different structures and wording while preserving the original meaning of the source text.

With the support of deep learning and Natural Language Processing (NLP) technology, this project represents a significant step forward in the development of Indonesian language–based artificial intelligence.
