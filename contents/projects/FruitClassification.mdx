## Overview

The **Fruit and Vegetable Classification System** is an end-to-end _Deep Learning_ academic project developed to demonstrate the capability of computer vision.

Built using _Keras_ and _TensorFlow_, this project utilizes the "Fruit and Vegetable Image Recognition" dataset from Kaggle. The primary objective is to build an intelligent model capable of automatically distinguishing between various classes of fruits and vegetables strictly based on raw image inputs.

## Architecture & Methodology

The core of this project relies on a custom-built **Convolutional Neural Network (CNN)**. The entire machine learning pipeline is structured logically to ensure optimal learning and generalization:

- **Image Preprocessing**:
  The raw image dataset undergoes rigorous preprocessing, including rescaling, resizing, and normalization, to ensure the data fed into the neural network is uniform and optimally structured.
- **CNN Architecture Development**:
  The model is constructed with multiple _Convolutional Layers_ to extract spatial features, followed by pooling layers to reduce dimensionality. This allows the network to learn intricate patterns, edges, and textures unique to each class of fruit and vegetable.
- **Training & Compilation**:
  The model is compiled and trained iteratively over **32 epochs**, allowing the network to continuously adjust its weights and minimize the _loss function_.

## Model Performance

The deep learning model achieved highly commendable metrics, proving its effectiveness and stability without severe _overfitting_:

- **Test Accuracy**: Reached **84.40%**, demonstrating a strong ability to classify completely unseen images accurately.
- **Validation Accuracy**: Reached **84.04%**, proving the model's stability throughout the training phase.
